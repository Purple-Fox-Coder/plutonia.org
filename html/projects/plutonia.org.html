Plutonia.org

This is my website where I post devlogs and blog a bit about interesting projects I'm working on or have seen! Topics will typically revolve around low level development, EEG research, and linux/OS development. I often prefer to use languages like C or Zig, but there may be a variety of languages here.

9/4/2025 - Beginnings

 Today was fairly simple, planning how I want to set up the site, what kind of things I want to post here.. the like. Not a ton to say, but I did find out about &emsp; and the & codes for html! This particular post is also doubling as a way to make sure I set up the styling and all correctly in the CSS which is fairly simple as of now. Here's a code block of it: 

          
            .blog {
              background: var(--colour-medium);
              min-height: 150px;
              margin-top: 5px;
            }

            .blog .section {
              padding: 20px;
              margin-top: 20px;
            }

            .blog h1 {
              padding: 30px;
              font-size: 40px;
              text-align: center;
              background: var(--colour-secondary-light);
            }

            .blog p {
              text-indent:initial;
            }

 I've set everything up so that I have the main site colours as variables for quick use. For this piece of styling in particular I mostly borrowed from w3schools' blog layout for simplicity. Since I intend to keep this website as static as possible (mostly for the sake of avoiding lots of hosting costs...) it's been an interesting challenge of figuring out how I want to lay things out so that it's still easy to navigate with minimal or ideally no JS usage (though I may add some in at some point for a simple search bar. I am also wondering if I might want subpages for the individual devlogs. I might do that for specific projects if their single page gets a little too long for my liking (maybe a prev and next buttons setup, to keep the simple feel).

 Today also marks the first thing posted here! :D (party noises) For my projects, they're going to be more of a devlog, I'm planning to add BLOG in the title of any actual blog/article posts I make here. Do I expect anyone to read these? Not exactly, but it seems like a fun project, and a nice way to get out my thoughts on various topics I'm interested in!
